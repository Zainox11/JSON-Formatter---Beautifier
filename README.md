# 🚀 Pro JSON Formatter, Validator & Minifier

A fast, secure, and fully client-side web application designed for developers to format, validate, and minify JSON data effortlessly. Built with a modern UI using Tailwind CSS.

## ✨ Features
- **Format & Beautify:** Instantly convert messy, unreadable JSON into perfectly indented code.
- **Minify JSON:** Compress JSON data by removing unnecessary whitespaces for production use.
- **Validate JSON:** Automatically detects syntax errors and highlights invalid JSON strings.
- **File Upload/Download:** Upload `.json` files directly from your computer and download the processed output.
- **One-Click Copy:** Easily copy the formatted or minified code to your clipboard.
- **100% Secure (Client-Side):** All processing happens locally in the browser. No data is ever sent to a server.

## 🛠️ Tech Stack
- **Structure:** HTML5
- **Styling:** Tailwind CSS (via CDN)
- **Logic:** Vanilla JavaScript
- **Icons:** FontAwesome
